◆ currentMessage()

template<typename T >
T eos::currentMessage ( )

This method attempts to reinterpret the message body as type T. This will only work if the message has no dynamic fields and the struct packing on type T is properly defined.

Example:

struct dummy_message {
char a; //1
unsigned long long b; //8
int c; //4
};
dummy_message msg = currentMessage<dummy_message>();

◆ requireNotice()

template<typename... Accounts>
void eos::requireNotice ( AccountName  name,
Accounts...  accounts 
)

All of the listed accounts will be added to the set of accounts to be notified

This helper method enables you to add multiple accounts to accounts to be notified list with a single call rather than having to call the similar C API multiple times.

Note
message.code is also considered as part of the set of notified accounts

Example:

requireNotice(N(Account1), N(Account2), N(Account3)); // throws exception if any of them not in set.
