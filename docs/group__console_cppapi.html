Detailed Description

This API uses C++ varidic templates and type detection to make it easy to print any native type. You can even overload the print() method for your own custom types.

### Example:

print( "hello world, this is a number: ", 5 );

Overriding Print for your Types

There are two ways to overload print:

  1. implement void print( const T& )
  2. implement T::print()const
