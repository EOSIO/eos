## Command

```sh
cleos wallet keys
```

## Description

Use this command to list all public keys stored in wallets. You can use these to sign transactions.


## Example


```console
cleos wallet keys
[[
    "EOS6MRyAjQq8ud7hVNYcfnVPJqcVpscN5So8BhtHuGYqET5GDW5CV",
    "5KQwrPbwdL6PhXujxW37FSSQZ1JiwsST4cqQzDeyXtP79zkvFD3"
  ]
]
```
